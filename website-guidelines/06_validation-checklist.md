# Validation Checklist — QA перед деплоем

Краткий чеклист для проверки лендинга перед публикацией.

---

## 1. Визуал и стили

- [ ] Палитра соблюдена (только цвета из `01_style-guide.md`)
- [ ] Нет SaaS-градиентов или левых цветов
- [ ] Все рамки 1px solid `--line` (hairline)
- [ ] Grain overlay на месте и не слишком сильный (opacity: 0.07)
- [ ] Изображения в grayscale, на hover — цветные
- [ ] Шрифты загружены корректно (Bebas Neue, Inter, Space Mono, Bodoni Moda)
- [ ] Responsive: проверь mobile и desktop (особенно грид и типографика)

---

## 2. Контент и текст

- [ ] Весь текст на русском (нет английских вставок)
- [ ] CTA везде одинаковый: "Анкета предзаписи"
- [ ] Нет запрещённых фраз из `03_copy-guide.md`
- [ ] Заголовки короткие и ударные (3-5 слов)
- [ ] FAQ: вопросы от пользователя, ответы конкретные
- [ ] Кейсы: метрики и факты (не "получил опыт", а "+15k подписчиков")
- [ ] Все placeholder'ы заполнены (нет "Lorem Ipsum")

---

## 3. Навигация и якоря

- [ ] Sticky nav на месте и работает
- [ ] Якоря совпадают: nav ссылки → секции IDs (`#about`, `#author`, `#cases`, `#faq`, `#presale`)
- [ ] Плавный скролл работает (smooth scroll в `main.js`)
- [ ] Sticky mobile CTA ведёт на `#presale`
- [ ] Footer ссылки рабочие (или заглушки)

---

## 4. Форма предзаписи

- [ ] Все поля обязательны: `name`, `phone`, `telegram`
- [ ] Чекбокс политики обязателен
- [ ] Плейсхолдеры корректны (`Arina`, `+7 (999) 000-00-00`, `username`)
- [ ] Submit кнопка: "Отправить заявку"
- [ ] Success message скрыт по умолчанию (`hidden`)
- [ ] После submit форма скрывается, показывается success message
- [ ] Валидация работает (проверь пустые поля)

---

## 5. Accessibility (A11y)

- [ ] `alt` атрибуты у всех изображений
- [ ] `aria-required="true"` на обязательных полях формы
- [ ] Focus ring на всех интерактивных элементах (акцентный цвет)
- [ ] Кнопки и ссылки имеют понятный текст (не "Кликни сюда")
- [ ] Контраст текста соблюдён (primary: `#F5F5F7` на `#0B0B0D` — ОК)

---

## 6. Медиа-файлы

- [ ] Все изображения в `site/assets/images/` (не file://)
- [ ] Пути относительные: `src="assets/images/фото Арины.jpg"`
- [ ] Нет битых ссылок на изображения
- [ ] Внешние CDN изображения загружаются (Unsplash, Supabase)

---

## 7. Скрипты и функциональность

- [ ] Lucide иконки загружаются (`lucide.createIcons()` в `main.js`)
- [ ] Smooth scroll работает
- [ ] Форма обрабатывается (`initPresaleForm()`)
- [ ] Активная навигация работает (опционально)
- [ ] Консоль браузера без ошибок (F12 → Console)

---

## 8. Производительность

- [ ] Tailwind загружается через CDN
- [ ] Custom CSS/JS подключены корректно
- [ ] Google Fonts загружаются с `preconnect`
- [ ] Нет дублирующих загрузок шрифтов
- [ ] Страница загружается < 3 сек (проверь в DevTools)

---

## 9. SEO и метаданные

- [ ] `<title>` заполнен: "AI FOR CREATORS 4.0 — Предзапись"
- [ ] `lang="ru"` в `<html>`
- [ ] `charset="UTF-8"` присутствует
- [ ] `viewport` мета-тег на месте
- [ ] (Опционально) Open Graph теги для соцсетей

---

## 10. Cross-browser тест

- [ ] Chrome: всё работает
- [ ] Safari: всё работает
- [ ] Firefox: всё работает
- [ ] Mobile Safari (iOS): всё работает
- [ ] Mobile Chrome (Android): всё работает

---

## Финальная проверка

- [ ] Прошёлся по всей странице (scroll top → bottom)
- [ ] Кликнул все ссылки и кнопки
- [ ] Заполнил форму и проверил submit
- [ ] Открыл FAQ (все аккордеоны)
- [ ] Проверил hover states (ссылки, кнопки, изображения)
- [ ] Ресайзнул окно (responsive)
- [ ] Нет опечаток в тексте

---

**Если всё ОК — деплой. Если нет — фиксим и проверяем снова.**
